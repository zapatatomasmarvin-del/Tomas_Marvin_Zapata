# Portafolio Ejecutivo - Estratega Digital

Este repositorio contiene el código fuente de mi **Landing Page Ejecutiva**, diseñada para presentar mis servicios como Estratega Digital, enfocados en Paid Media, Automatización, Producción Comercial y SEO en Marketplaces.

## Características del Proyecto

- **HTML5 Semántico**: Estructura limpia y optimizada para SEO.
- **CSS Vanilla (Glassmorphism & Dark Mode)**: Diseño estético y de alto impacto.
- **Animaciones al Scroll**: Interacciones suaves para una experiencia de usuario premium.
- **Totalmente Responsive**: Adaptado para Desktop, Tablets y Móviles.

## Estructura de Archivos

- `index.html` - Archivo principal que contiene toda la estructura de la web y el código CSS.
- `assets/` - Carpeta que contiene las imágenes de los Casos de Éxito (`zafiro.png`, `bossburgers.png`, `accesopadel.png`).

## Despliegue en Vercel

Este proyecto está listo para ser desplegado instantáneamente en [Vercel](https://vercel.com).
Como es un sitio web estático puro (Vanilla HTML/CSS), Vercel no necesita ninguna configuración adicional.

1. Sube este código a un nuevo repositorio en tu GitHub.
2. Ingresa a Vercel, haz clic en **"Add New"** > **"Project"**.
3. Conecta tu cuenta de GitHub e importa este repositorio.
4. Dale a **"Deploy"**. Vercel detectará el archivo `index.html` automáticamente.
